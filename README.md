# Prodomart: The Product Vessel

### THE ANTI-MASS STANDARD.
> "We do not aggregate for the masses. We index for the few. No smooth corners. No forgiving algorithms. Just absolute utility."

Prodomart is a minimalist, industrial-inspired product showcase designed with a brutalist aesthetic. It serves as a strict conduit for physical artifacts, prioritizing high-contrast utility and raw digital craftsmanship.

![Prodomart Showcase](https://images.unsplash.com/photo-1550745165-9bc0b252726f?auto=format&fit=crop&q=80&w=1200)

## 🛠 Features

- **Brutalist UI/UX**: A raw, uncompromising design system built on custom CSS properties and rigid layouts.
- **3D Interactive Gallery**: A custom-built 3D cover flow animation using CSS `view-timeline` and `transform-style: preserve-3d`.
- **Integrated Product Search**: Instant, tag-based filtering for finding artifacts across different archives.
- **Night Mode**: A high-utility dark mode toggle (click the logo to activate).
- **Strict Inquiries**: Two-track inquiry system (Order / Submission) with real-time validation and automated email transmission.
- **Responsive Architecture**: Built for high-density displays and mobile terminals alike.

## 🚀 Tech Stack

- **Framework**: [Vite](https://vitejs.dev/)
- **Core**: Vanilla JavaScript (ESM)
- **Styling**: Vanilla CSS3 (Custom Variables, CSS Grid, Flexbox)
- **Typography**: Fraunces (Headings), Roboto Mono (Body)

## 📦 Setup & Installation

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/brutalist-showcase.git
   cd brutalist-showcase
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 📐 Design Ethos

The project follows the principles of **Brutalist Web Design**:
- RAW content and minimal decoration.
- HIGH contrast color palettes.
- RIGID, predictable layouts.
- ABSOLUTE transparency of function.
